🚀 React 30 Projects Learning Repository

A complete React.js learning roadmap from Beginner → Advanced through 30 practical projects.

This repository is designed for developers who want to master React by building real projects, not just reading theory.

Each folder contains an independent React project focusing on a specific concept such as:

React Hooks

Component Architecture

API Integration

State Management

Authentication

Dashboards

Real-time apps

Full-stack integration

Clone the repository

git clone https://github.com/riyazwani97/react-30-projects-learning.git

Go to any project folder

cd 01-counter-app

Install dependencies

npm install

Start the project

npm start
